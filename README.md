

# StudyTravel Simulator

A gamified flight-simulator focus timer that turns study sessions into journeys.

**Created by:** tekulapalli abhiram (@pokephantom98765)

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` in `.env.local` (or environment) to your Gemini API key.
3. Run the app:
   `npm run dev`

## Features

- Gamified study timer with flight simulator theme
- Journey tracking and analytics
- Interactive sky visualization
- Session feedback system
- Firebase backend with security rules

## Production Preview

Production preview (serves `dist/` and API) can be run with:
```
npm run preview
```
